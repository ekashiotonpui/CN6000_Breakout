ATH530 Breakout Simulator
1. 概要
本シミュレーターは、Node.js などのサーバー環境を必要とせず、ブラウザ（index.html）を直接開くだけで動作するシングルページアプリケーションです。

2. ファイル構成
index.html: 本体。HTML/CSS/全シナリオデータ/JavaScript すべてを内包しています。
assets/: 画像フォルダ。セキュリティとデータ保護のため、画像ファイルはすべて拡張子を .dat に変換して格納されています。

起動はプロジェクトフォルダーで Powershellを起動し、 node server.js でサーバー起動。
http://localhost:3000 で開発を行う
サーバー終了はCtrl+C

3. シナリオデータの編集方法 (SCENARIO_DATA)
index.html 内の <script> タグ冒頭にある SCENARIO_DATA オブジェクトを直接編集します。
start_scene: 起動時に最初に表示されるシーンIDを指定します。
scenes: 各シーンの設定（画像名、タイトル、ホットスポット等）を記述します。
rect: [top, left, width, height] の順に、画像に対するパーセンテージで指定します。
target_scene: クリック時の移動先となるシーンID（scenes 内のキー名）を指定します。

4. 特殊操作（開発者向け機能）
隠しコマンド: Shift + Alt + E を同時に押すと、画面下部に「Edit Mode」パネルが表示されます。

Edit Mode:
パネル内のチェックボックスを ON にすると編集モードになります。
編集モード中は、ホットスポットをクリックしても遷移せず、コンソールに座標やインデックスが表示されます。
右クリック禁止: 通常時は右クリックによる画像保存を禁止していますが、Edit Mode が表示されている間のみ右クリックが有効化されます。

5. 注意事項
画像参照: getAsset 関数により、どんな拡張子を指定しても自動的に assets/ フォルダ内の .dat ファイルを探しに行きます。
エラー対応: 万が一画面が真っ白になった場合は、F12 キーでコンソールを開き、SCENARIO_DATA の閉じカッコ }; が正しく閉じられているか確認してください。
